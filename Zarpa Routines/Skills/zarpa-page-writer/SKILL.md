## WE TREAT MODE — Сторінка "Ми лікуємо"

Мета: пацієнт або батьки шукають інформацію про хворобу/стан → знаходять zarpa.io →
розуміють стан дитини/пацієнта → бачать що Zarpa лікує це → записуються.

Це контентна, освітня сторінка про стан — не рекламна сторінка послуги.
Читач хоче зрозуміти: "що це таке?", "як це лікують?", "чи підходить Zarpa?"

### Структура We Treat Mode

**① Hero-блок**
```
H1: [Назва стану/хвороби] — [коротка обіцянка результату або ключовий факт]
Мета-відповідь (2-3 речення): пряма відповідь для AEO / Google snippet
CTA: "Записатись на консультацію"
```
Приклади H1:
- "Сколіоз у дітей — як виправити поставу без операції"
- "Біль у спині: причини та відновлення за допомогою фізичної терапії"
- "Кривошия у немовлят — лікування від 1 місяця"

**② Що таке [стан] — визначення**
```
H2: Що таке [стан]
```
Просто і зрозуміло. Без медичного жаргону або з поясненням відразу.
2-4 абзаци. Для дитячих станів — пояснення для батьків, не лікарів.

**③ Причини та фактори ризику**
```
H2: Причини [стану] та хто в групі ризику
```
Список або абзаци. Конкретно: вік, спосіб життя, попередні травми.

**④ Симптоми та ознаки**
```
H2: Як розпізнати [стан]: симптоми і ознаки
```
Ознаки які видно батькам / відчуває пацієнт — без складних термінів.
Якщо є візуальні ознаки — описуй: "одне плече вище іншого", "дитина нахиляє голову вправо".

**⑤ Методи фізичної терапії у Zarpa**
```
H2: Як у Zarpa лікують [стан]
```
НЕ перелік послуг. Пояснення підходу: що робить фізичний терапевт, які техніки, який курс.
Конкретно: "перші 2-3 сеанси діагностика і оцінка", "курс зазвичай X-Y сеансів".
Не обіцяти конкретних результатів — "у більшості випадків", "залежно від стадії".

**⑥ Протипоказання (обов'язково)**
```
H2: Коли потрібна консультація перед фізичною терапією
```
Чесно про випадки де потрібна спочатку консультація лікаря або де ФТ не підходить.
Підвищує E-E-A-T і довіру.

**⑦ Блок Zarpa (E-E-A-T)**
```markdown
## Фізична терапія при [стан] у Zarpa — Львів

Центр Zarpa працює з [стан] з [рік]. Наші фізичні терапевти...
[Ім'я фахівця] — [посада "фізичний терапевт"], [роки досвіду], [спеціалізація].
Медична ліцензія. Два відділення у Львові.

Запишіться на первинну консультацію → [CTA]
```

**⑧ FAQ (мінімум 6 питань)**
```
H2: Часті запитання про [стан]
```
Питання — як люди питають у Google і в пошукових підказках.
Для дитячих тем: питання від батьків ("Чи болісно?", "Скільки сеансів?", "З якого віку?").
Формат: H3 + відповідь 2-5 речень.

**⑨ CTA**
```markdown
Маєте запитання або хочете записатись?
Залиште контакт — підберемо зручний час і відповімо на ваші питання.
[Записатись на консультацію]
```

### Правила We Treat Mode
- H1 містить назву стану + пошуковий намір (виправити, лікувати, допомогти, терапія)
- "у Львові" в H1 та мета-тегах
- Перші 2-3 речення після H1 = пряма відповідь (AEO / Google snippet)
- Структуру можна адаптувати під конкретний стан: рідкісний стан потребує більше пояснення, відомий — можна скоротити визначення
- Ніколи: "Ми пропонуємо", "Наша клініка надає" — писати від перспективи пацієнта
- Протипоказання обов'язкові
- Вигадана статистика заборонена — додай мітку `[потрібна перевірка]` якщо не певен у цифрі

### Чекліст We Treat Mode
- [ ] H1 з назвою стану + "у Львові"
- [ ] Мета-відповідь 2-3 речення (AEO snippet)
- [ ] Визначення стану зрозуміло для батьків/пацієнтів
- [ ] Причини та фактори ризику
- [ ] Симптоми (візуальні ознаки де можливо)
- [ ] Методи ФТ у Zarpa (без гарантій)
- [ ] Протипоказання
- [ ] Блок Zarpa з іменем фахівця
- [ ] FAQ мінімум 6 питань
- [ ] CTA
- [ ] SEO Title + Meta Description
- [ ] img_prompt заповнений (в custom field через ClickUp API)

## LANDING MODE — Сторінка Послуги

Мета: пацієнт шукає послугу у Львові → знаходить zarpa.io → розуміє що Zarpa підходить → записується.

### Структура

**① Hero-блок**
```
H1: [Назва послуги] у Львові — [коротка ціннісна пропозиція]
Підзаголовок: [1-2 речення: для кого + що отримає]
CTA: "Записатись на консультацію"
```
Приклад:
```
H1: Масаж для немовлят у Львові — розвиток і спокій з перших місяців
Підзаголовок: Курс дитячого масажу від сертифікованого фізичного терапевта.
              Для дітей від 1 місяця. Центр Zarpa, два відділення.
```

**② Для кого ця послуга**
```
H2: Кому підходить [послуга]
```

**③ Що отримає пацієнт**
```
H2: Що дає [послуга]
```
Не "ми робимо X" — а "ви отримаєте X". Конкретно, без порожніх прикметників.

**④ Як проходить**
```
H2: Як проходить [послуга] у Zarpa
```
Покроково: перший прийом → сеанс → скільки часу → скільки сеансів зазвичай.

**⑤ Наш фахівець (E-E-A-T)**
```
H2: Хто проводить [послугу]
```
Ім'я, посада "фізичний терапевт", роки досвіду, спеціалізація. Медична ліцензія.

**⑥ Протипоказання (обов'язково)**
```
H2: Коли [послуга] не підходить
```

**⑦ Вартість і запис**
```
H2: Вартість і як записатись
```

**⑧ Локація**
```
H2: Де ми знаходимось
```
Обидві адреси, Львів.

**⑨ FAQ (мінімум 5 питань)**

### Правила Landing Mode
- Жоден абзац не починається з "Ми пропонуємо" або "Наш центр надає"
- Замість: "Ви отримаєте", "Ваша дитина", "Пацієнти Zarpa..."
- "у Львові" в H1 і мета-тегах
- Без суперлативів: "найкращий", "унікальний" — ні
- Протипоказання обов'язкові

### Чекліст Landing Mode
- [ ] H1 з "у Львові"
- [ ] Блок "для кого"
- [ ] Процес покроково
- [ ] Фахівець з кваліфікацією
- [ ] Протипоказання
- [ ] FAQ мінімум 5 питань
- [ ] CTA на запис
- [ ] SEO Title + Meta Description

## MARKETING MODE — Маркетингова Сторінка

Мета: емоція → бажання → дія. Менше інформації, більше відчуття. Обсяг 400–700 слів.

### Структура

**① Потужний перший рядок (гачок)**
- Питання: "Коли ви востаннє прокинулись без болю?"
- Контраст: "Спина не мусить боліти. Ніколи."
- Провокація: "Ви знову відкладаєте?"

**② Картина болю (проблема)**
2-3 речення. Читач впізнає себе. Без діагнозу — тільки ситуація.

**③ Картина результату**
Як виглядає життя після. Конкретно і образно.

**④ Чому Zarpa**
3-5 коротких пунктів. Факти: "фізичні терапевти з X роками досвіду", "медична ліцензія".

**⑤ Соціальний доказ (якщо є)**
Цитата пацієнта (анонімно або за згодою), 2-3 речення.

**⑥ Один чіткий CTA**
```
[Записатись на безкоштовну консультацію]
```

### Правила Marketing Mode
- Абзаци: 2-3 речення максимум
- Один CTA — не розпилюватись
- Порядок: Емоція → Логіка → Дія

### Чекліст Marketing Mode
- [ ] Перший рядок — гачок
- [ ] Читач впізнає себе
- [ ] Один CTA
- [ ] Обсяг не більше 700 слів
- [ ] SEO Title + Meta Description
