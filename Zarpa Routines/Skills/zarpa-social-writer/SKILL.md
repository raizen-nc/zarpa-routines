# zarpa-social-writer — Соціальні Мережі Zarpa

Перед початком переконайся що завантажив:
- `zarpa-brand-voice` — тон і термінологія
- `zarpa-seo-aeo-framework` — (тільки SEO Title / Meta якщо пост дублюється на сайт)

---

## 1. Визначення Формату

З таску витягни:
- Платформа: Instagram / Facebook / TikTok / All (якщо не вказано — писати для Instagram)
- Формат: Reels caption / Carousel / Static post / Stories текст / Saved post
- Тема і аудиторія

**Якщо формат не вказаний — писати Instagram Static / Carousel за замовчуванням.**

---

## 2. Формати Постів

### 📱 Instagram Static або Facebook Post

Структура:
```
[ГАЧОК — перший рядок]

[Тіло — 3-5 коротких абзаців, 2-3 речення кожен]

[CTA — 1 рядок]

[Хештеги — окремим блоком]
```

**Ліміт:** 150-300 слів тіло + хештеги.

---

### 🎠 Carousel (навчальний або інформаційний)

Структура слайдів:
```
Слайд 1: Заголовок — гачок або питання
Слайд 2–N: Один факт/крок/правило на слайд (1-3 речення)
Останній слайд: CTA + посилання в bio
```

**Ліміт:** 5-8 слайдів. Кожен слайд — одна думка.
Caption під постом — коротке (50-80 слів) + хештеги.

---

### 🎬 Reels Caption

Структура:
```
[1 рядок-гачок — максимально коротко]
[2-3 речення що у відео]
[CTA]
[Хештеги]
```

**Ліміт:** 80-120 слів.

---

### 💾 Saved Post (корисний пост для збереження)

Структура:
```
[Заголовок: "Збережи — знадобиться"]
[Основний контент: список, покроково або питання-відповідь]
[CTA: "Поділись з тим хто потребує"]
```

**Ліміт:** 200-400 слів.

---

## 3. Гачки — Банк Форматів

Обирай відповідний під тему:

**Питання до аудиторії:**
- "Ваша дитина скаржиться на спину після школи?"
- "Коли ви востаннє прокинулись без болю?"
- "Знаєте яку помилку роблять 9 з 10 батьків при болях у ніг дитини?"

**Контраст:**
- "Спина не мусить боліти. Ніколи."
- "Норм — це коли рухаєшся вільно. Не коли терпиш."

**Провокація / Тренди:**
- "POV: ти нарешті записався до фізичного терапевта"
- "Норм чи стрьом: масаж дитині до 1 року?"
- "Що буде якщо ігнорувати біль у спині 3 місяці?"

**Факт або статистика (тільки з перевіреного джерела):**
- "Сколіоз у дітей діагностують найчастіше між 10 і 14 роками."

**Від терапевта (від першої особи):**
- "На першому прийомі мене запитують одне і те саме..."
- "За 8 років практики я бачив це сотні разів."

---

## 4. Тон для Соціальних Мереж

З `zarpa-brand-voice` — "Молода команда":
- Живий, розмовний, але компетентний
- Перша особа терапевта АБО центру — залежно від задуму
- Гумор дозволений — без глузування з болю або будь-якої групи
- Emoji — помірно (1-3 на пост), завжди по темі
- Трендові формати: "Норм чи стрьом?", "POV:", "Що буде якщо...", "Чи можна..."

---

## 5. Хештеги

**Базові для Zarpa (додавати завжди, 3-5 з цього списку):**
```
#zarpa #zarpaлвів #фізичнатерапія #фізичнийтерапевт #реабілітаціяльвів
```

**Тематичні (2-4 під конкретний пост):**

Дитячі теми:
```
#дитячамасаж #педіатричнатерапія #сколіоздітей #розвитокдитини #кривошиядитини
```

Дорослі / спина:
```
#більуспині #реабілітація #постава #здоровяспини #болюколінах
```

Загальні охоплення:
```
#льів #здоровялвів #фізіотерапіяльвів #реабілітаціяукраїна
```

**Загальна кількість хештегів:** 8-15 (для Instagram), 3-7 (для Facebook).

## 6. Спеціальні Формати — "Рубрики"

Якщо в описі задачі згадується рубрика — використовуй її стиль:

| Рубрика | Стиль |
|---------|-------|
| "Запитай терапевта" | Від першої особи терапевта, Q&A |
| "Міф vs Факт" | Carousel, 1 міф = 1 слайд, розбиваємо |
| "До / Після" | Лаконічно, з фокусом на функцію (не зовнішність) |
| "День у Zarpa" | Сторітелінг, легкий тон, за лаштунками |
| "Поради для дому" | Saved post, покроково, практичні вправи |

---

## 7. Заборонено в Соціальних Мережах

- Фото або опис больових процедур без попередження
- До/Після фото без явної згоди пацієнта
- Медичні обіцянки ("вилікуємо", "позбавимо назавжди")
- Залякування ("якщо не лікувати — стане гірше")
- Порівняння з конкурентами по імені
- Контент про аутизм, психічні розлади
- Будь-які політичні коментарі

## 9. Фінальна Перевірка

- [ ] Перший рядок — гачок (не "Сьогодні ми розповімо...")
- [ ] Тон — жива мова, не корпоративний шаблон
- [ ] Жодного "фізіотерапевт" — тільки "фізичний терапевт"
- [ ] Emoji є, але не більше 3-4 в тексті
- [ ] CTA чіткий і один
- [ ] Хештеги базові Zarpa + тематичні
- [ ] Немає медичних обіцянок
- [ ] Формат відповідає платформі (Instagram vs Facebook vs Reels)
